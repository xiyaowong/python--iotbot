# 安装

## 环境

python 版本: 3.6+

## 在线安装

```shell
pip install git+https://github.com/XiyaoWong/python-iotbot.git@master
```

```shell
pip install python-iotbot
```

推荐使用第一种，pypi 上的版本滞后

## 离线安装

```shell
git clone https://github.com/XiyaoWong/python-iotbot
cd python-iotbot
python setup.py install
```
